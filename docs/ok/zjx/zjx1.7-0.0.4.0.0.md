### 四、外科症状

治疗外科病症，多取邻近穴及与该部位有经络联系的远道穴。

疔肿在面部：针灸合谷、足三里、神门。

疔肿在手部：灸曲池。

疔肿在背部：针灸肩井、足三里、委中、足临泣、行间、通里、少海、太冲，灸骑竹马穴。

臁疮：针血海。

乳房肿痛：针足临泣。

乳房红肿：针膺窗、乳根、下巨虚、复溜、太冲。

肠痈腹痛：针上巨虚、天枢、地机、阑尾。

瘰疬：针肩井、曲池、天井、三阳络、阳陵泉。

瘿气：针气舍、间使、太冲、太溪、臑会。

脱发：针阿是穴、百会、风池、膈俞、足三里、三阴交。

腰部疱疹：取患部邻近的穴位及身柱、合谷、曲池、血海、三阴交。

皮肤风疹：针血海、三阴交、曲池、合谷，或针大肠俞。
