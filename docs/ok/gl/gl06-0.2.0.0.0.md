## 第二节  气血津液辨证

气血津液辨证，就是分析气、血、津液各方面的病理变化，从而辨识其所反映的不同证候的一种辨证方法。

气血津液病变与脏腑病变密切相关，二者常常相互影响，互为因果。气血津液辨证是对脏俯病变中气血津液失常的某些共同病理特征的概况，因而，掌握这一辨证，可为学习脏腑辨证奠定基础。

气血津液辨证包括气病的辨证、血病的辨证、气血同病辨证，与津液病的辨证。现分列简表如下：
