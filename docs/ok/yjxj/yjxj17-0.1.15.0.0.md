### 陈修园医书十六种

清•陈修园（名念祖）撰，刊于1865年。

本书又名《南雅堂医书全集十六种》，包括以下十六种：

1.《灵素节要浅注》十二卷。

2.《金匮要略浅注》十卷。

3.《金匮方歌括》六卷。

4.《伤寒论浅注》六卷。

5.《长沙方歌括》六卷。

6.《医学实在易》八卷。

7.《医学从众录》八卷。

8.《女科要旨》四卷。

9.《神农本草经读》四卷。

10.《医学三字经》四卷。

11.《时方妙用》四卷。

12.《时方歌括》二卷。

13.《景岳新方砭》四卷。

14.《伤寒真方歌括》六卷。

15.《伤寒医诀串解》六卷。

16.《十药神书注解》一卷。

本书以1906年上海飞鸿阁书局石印本较为常见。
